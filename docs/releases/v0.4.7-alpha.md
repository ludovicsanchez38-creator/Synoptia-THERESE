### BUG-069 - Email "account not found" après reconfiguration
- **Symptôme** : bandeau rouge "Échec du rafraîchissement : Email account not found" après avoir reconfiguré un compte Gmail
- **Cause racine** : le store Zustand (persist localStorage) gardait l'ancien `currentAccountId` en cache. Après reconfiguration, le nouveau compte a un nouvel ID, mais le store envoyait toujours l'ancien → 404 backend.
- **Fix** : `EmailPanel.tsx` vérifie si le `currentAccountId` existe encore dans la liste des comptes renvoyés par l'API. Sinon, bascule sur le premier compte disponible.

### BUG-070 - Rate limiting Gmail enrichment
- **Symptôme** : enrichissement des emails échoue partiellement (certains messages avec `error` au lieu des métadonnées)
- **Cause racine** : `asyncio.gather` lançait 50 requêtes `get_message()` en parallèle → rate limiting Gmail API + race condition token refresh
- **Fix** : `email.py` utilise un `asyncio.Semaphore(5)` pour limiter à 5 requêtes simultanées

### UI - Icône Board 24px
- `ChatHeader.tsx` : icône Board passée de 18px à 24px dans la barre du haut
