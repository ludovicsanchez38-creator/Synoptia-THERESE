## Audit

Mode complet, au-delà du standard : chaque sprint a subi une **revue
adversariale par workflow de 4 sceptiques** avant son merge (pas un audit de
PR à 3 agents - le contenu était déjà sur main, audité par sprint).

Bilan des revues : **6 findings critiques** (dont 4 démontrés par repro
exécutée) et ~18 importants, tous corrigés avant merge :
- chat Ollama cassé sur les modèles sans tools (chaque message en 400) ;
- message d'erreur « honnête » qui était du code mort (ResponseNotRead) ;
- backup non restaurable sans la clé de chiffrement (disaster recovery mort) ;
- 3 jobs CI morts-nés (mypy tué par pipefail, rust sans dist, pip-audit rouge).

Décision Claude Code : **MERGER** - suites complètes vertes à chaque étape.
