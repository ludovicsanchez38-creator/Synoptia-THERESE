## Incident build

Le premier build Linux a échoué : le pattern glob `binaries/backend-libs/**` n'est pas supporté par Tauri (le `**` seul ne match pas les fichiers). Corrigé en revenant aux deux patterns explicites `binaries/backend-libs/*` et `binaries/backend-libs/_internal/**/*`. Tag supprimé et recréé. Deuxième build : 3/3 success.

## Warnings et dette technique

- Le `catch` dans SettingsModal.tsx utilise un cast explicite `as Record<string, boolean>` pour contourner l'inférence TypeScript - acceptable mais pas idéal
- Le helper `_check_key_decryptable` fait un appel DB synchrone par clé (11 appels séquentiels) - pourrait être optimisé en batch
- Le BUG-052 fix dans `get_llm_service_for_provider()` crée une connexion DB manuelle au lieu d'utiliser la session existante - refacto souhaitable
