## Changements

### BUG-049 — Bouton "Re-tester" Ollama
- `LLMTab.tsx` : prop `onRetestOllama` + bouton `RefreshCw` avec animation spin
- `SettingsModal.tsx` : fonction `retestOllama()` + état `retestingOllama`
- L'utilisateur peut re-tester Ollama sans fermer/rouvrir les paramètres

### BUG-050 — Timeout skills Office sur machines lentes
- `ollama.py` : `timeout=120s` remplacé par `httpx.Timeout(connect=5.0, read=None)`
- Le `connect=5.0` détecte rapidement si Ollama n'est pas démarré
- Le `read=None` permet des générations longues sur CPU lent (Pentium G620 + gemma3:1b)
- Catch `ReadTimeout` retiré (code mort avec `read=None`)

### BUG-Gmail-403 — Erreur 403 access_denied Gmail
- `email.py` : détection `error == "access_denied"` → page HTML explicative (3 causes + solutions)
- `VerifyStep.tsx` : bloc `<details>` "Google affiche une erreur 403 ?" visible pendant l'attente
- Corrections XSS : `html.escape()` sur `email_address` et `str(e)` dans les pages OAuth

### Test adapté
- `test_regression.py` : test BUG-031 adapté au tri déterministe `Message.id.desc()` (commit Ludo PR #21)
