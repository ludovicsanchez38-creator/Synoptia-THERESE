## Ce qui change (pour les testeurs)

- **Confiance / souveraineté** : fini le bluff sur l'hébergement (distinction stockage local vs traitement cloud) ; souveraineté **honnête sur le chiffrement** (la base SQLite n'est pas chiffrée au repos, seuls les secrets le sont via Fernet AES-128) ; badge local/cloud par message ; RAG juridique ancré sur des références vérifiées.
- **CRM / facturation** : scoring réparé, notes persistées, profil émetteur + garde-fou facture, **devise étrangère honnête jusque dans le PDF** (plus de 40 EUR / L441-10 sur une facture CAD).
- **Email / documents** : éditeur de signature mail par compte, génération de vrais fichiers Word/PPT/Excel.
- **Anti-hallucination** : lecture de la vraie fiche contact, absence dite (calendrier), purge de l'index à l'anonymisation RGPD.
- **Navigation** : vues content-swap, palette ⌘K, Échap unifié, pastille contacts liés.
- **Multi-LLM / Board** : **outils Mistral réparés** (plus de réponse vide sur CRM / Word / board) ; board robuste au rate-limit (concurrence limitée par fournisseur).
