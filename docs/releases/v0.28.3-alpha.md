## Gotchas de session (pour les prochaines releases)

- **pytest silencieux** : le hook `os._exit` du conftest avale le résumé final
  ET les erreurs de collection (rien ne s'affiche avant le summary quand la
  collection échoue). En cas de « exit 2 sans aucune sortie » : lancer sous
  `script -q` (pseudo-TTY) et penser à `--ignore=tests/e2e` (le Makefile
  l'exclut, la collection e2e casse hors CI).
- **Dupliqués macOS** : 4 fichiers `tests/test_* 2.py` (artefacts Finder/iCloud,
  même famille que les refs git « doublon avec espace » du 09/07) cassaient la
  collection locale - déplacés à la corbeille, surveiller leur réapparition.
- **mypy local** : utiliser l'invocation exacte du CI
  (`--ignore-missing-imports --no-error-summary`, compter les ` error:`) -
  sans le flag, +465 erreurs d'imports fantômes.
