# Release v0.6.0-alpha - Rattrapage Manus + Quick Wins UX

**Date** : 7 mars 2026
**Version** : 0.5.4 -> 0.6.0
**Mode** : Release directe (branche feature, pas de PRs Zezette)

## Contexte

Dr_logic-3D (Jerome) a compare THERESE a Manus (agent IA Meta) dans le Discord alpha.
Il a identifie des fonctionnalites manquantes (browser, deep research, providers LLM).
Cette release comble les ecarts principaux et ajoute des quick wins UX demandes par les testeurs.

## Contenu

### 1. Providers LLM natifs : Perplexity + DeepSeek
- 2 nouveaux providers OpenAI-compatibles
- Perplexity : 4 modeles Sonar (recherche web integree)
- DeepSeek : 2 modeles V3 + R1 (raisonnement, prix bas)
- UI Settings + Onboarding + EnvVarModal

### 2. Quick Wins UX (5 items)
- Conversations vides : lazy save, persistance au 1er message seulement
- Renommer conversations : double-clic + menu contextuel dans la sidebar
- Labels OAuth en francais : "ID client" / "Code secret du client"
- Recommandation modele Ollama : 4 categories (general/coding/writing/fast)
- Raccourcis clavier adaptes Windows/Mac (Ctrl+K au lieu de Cmd+K sous Windows)

### 3. Deep Research
- Recherche approfondie multi-sources via Brave Search
- Decomposition LLM en 5-8 sous-requetes paralleles
- Synthese structuree avec citations [1], [2]...
- Bouton loupe cyan dans le chat

### 4. Skill HTML (Page Web)
- Generation de pages web completes (landing, portfolio, mini-site)
- Palette adaptee au sujet (pas de branding fixe)
- Bouton "Page Web" dans les Guided Prompts

### 5. Browser Automation v1
- Service Playwright headless (navigate, click, fill, extract, screenshot, links)
- Router /api/browser avec 9 endpoints
- Tool LLM browser_navigate integre au chat (tool calling)
- Securite : URLs http/https uniquement, timeout 30s

### 6. Bugs Dr_logic-3D (7 mars)
- OpenRouter crash ValueError: Invalid format string (accolades dans profil/THERESE.md)
- Logo Mac affiche sous Windows (raccourcis Cmd+K au lieu de Ctrl+K)
- Lien Brave Search ne s'ouvre pas sous Firefox/Windows (Tauri shell open)

### 7. Infrastructure tests
- THERESE_SKIP_SERVICES dans le lifespan pour les tests de routers
- test_regression.py restaure (4116 lignes)
- Test BUG-053 adapte pour .replace() au lieu de .format()
