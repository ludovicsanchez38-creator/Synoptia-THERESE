## Détails techniques

### BUG-060 - Fallback Ollama + UX Board
- Board souverain : cartes dépliables au clic, plus compactes par défaut

### BUG-061 - Refresh emails (fix majeur)
**Problème** : ~150 appels API par clic refresh (51 Gmail séquentiels + 50 fetch redondants + 50 LLM bloquants)
**Solution triple** :
1. Backend : `asyncio.gather` pour enrichissement concurrent (51 appels → ~2s au lieu de ~25s)
2. Frontend : mapping direct depuis les données list (suppression de 50 `getEmailMessage` redondants)
3. Frontend : classification IA en background (non bloquante) via `updateMessage` incrémental
- Ajout de `is_read`/`is_starred` dans la réponse backend
- Erreurs affichées en toast 4s même avec cache (plus de silence)

### BUG-062 - Timeout MCP init
**Problème** : `npx -y @modelcontextprotocol/server-google-workspace` télécharge le package au premier lancement, dépassant le timeout 30s
**Solution** : paramètre `timeout` ajouté à `_send_request()`, initialisé à 90s pour `_initialize_server`
