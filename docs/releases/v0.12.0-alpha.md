## Contenu (refonte direction artistique)

- **Système de thème clair / sombre** : mode clair « Équilibre » par défaut, mode sombre « Signature » disponible via la bascule (Réglages > Accessibilité). Tokens CSS theme-aware dans `globals.css` (`@theme` + `[data-theme="dark"]` + contraste élevé).
- **Tableau de bord « Ma journée » refondu** selon la maquette Claude Design : en-tête (salutation, date, recherche, pills En ligne / Données locales), 4 indicateurs colorés (k1-k4), panneaux Agenda + « Thérèse vous suggère », barre de saisie.
- **Tous les écrans adaptés aux deux thèmes** : conversation, sidebar, réglages, board des 5 conseillers, atelier, CRM, factures, projets/mémoire, tâches, calendrier, email. Couleurs sombres codées en dur remplacées par les tokens (la nav et la structure n'ont pas changé).
- **Correction de tokens no-op pré-existants** : `bg-background` (179 usages) et `text-text-primary` (9 usages) référençaient des tokens jamais définis dans `@theme` (fonds transparents, couleur héritée). Définis dans les trois blocs de thème : les inputs et panneaux insérés ont enfin un fond adaptatif.
- **Accents** : libellés et notifications affichés corrigés partout (À faire, Terminé, Découverte, Annulé, facture créée / payée / mise à jour, étapes, mentions RGPD...). Clés techniques et commentaires laissés intacts.
