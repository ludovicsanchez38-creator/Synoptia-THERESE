- **CC/CCI** dans le composeur email (toggle Cc/Cci, champs Cc et Cci)

- **Sanitizer HTML** (nh3) : whitelist tags/attributs pour signatures email
- **Sécurité** : instruction négative CRM dans response generator, HIGH_VALUE_SENDERS resserré

### Documentation
