## Point de vigilance (BUG-128)

Le code de l'updater suit le pattern officiel Tauri v2 (`downloadAndInstall` ->
`relaunch`, NSIS passif `/P /R`). Le correctif livré porte sur **l'affichage**
(fini l'alerte trompeuse). **L'auto-restart Windows lui-même reste à valider sur
une vraie machine Windows** : non reproductible depuis macOS. Un test updater
0.28.1 -> 0.28.2 par un testeur Windows confirmerait le comportement réel.
