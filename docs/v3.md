## Images

### US-IMG-08 : Gestion des images stockées

**Problème** : Les images générées s'accumulent dans `~/.therese/images/` sans nettoyage automatique.

**Solution proposée** :
- Nouvel onglet "Images" dans Settings
- Liste des images générées avec preview miniature
- Métadonnées : date, taille, provider, prompt
- Actions :
  - Supprimer une image
  - Nettoyer images > 30 jours
  - Nettoyer tout
- Affichage espace disque utilisé
- Option : compression automatique des PNG (optipng/pngquant)

**Complexité** : Medium
**Impact** : Low (nice-to-have)
